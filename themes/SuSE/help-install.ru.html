Справка по Загрузчику

Вас приветствует ∏uct;

Используйте это меню для выбора необходимой функции. Если у вас есть проблемы с навигацией в системе помощи, нажмите F1 что просмотреть описание системы помощи. Основные функции в этом меню:

Загрузка с жесткого диска: Этот выбор ничего не делает с системой. Она только загружает уже ранее установленную оперционную систему.

Установка: Этот режим установки работает в большинстве случаев. Если ваша система зависает при загрузке или есть проблемы при обнаружении вашего оборудования, такого как контроллеры диска или сетевые карты, попробуйте один из следующих параметров установки.

Установка -- ACPI отключено: Многие из проданных компьютеров в настоящее время имеют неточную реализацию ACPI функций. Здесь вы можете отключить поддержку ACPI в ядре, но по прежнему использовать остальные особенности, например DMA для жестких дисков IDE.

Установка -- Безопасные настройки: Если у вас не удалось провести установку при помощи Установка, выбрав этот пункт, вы можете попробовать решить ваши проблемы.

Спасательная система: Этот загрузочный образ запустит маленькую Linux систему в RAM. Это бывает нужным, если система не запускается. После загрузки, войдите в систему как суперпользователь.

Тест памяти: Проверка памяти полезна для дополнительной проверки после установки новых модулей памяти. Это нагрузочный тест основной части вашего компютера и может вызвать сбой.

Параметры загрузки: Параметры загрузки могут полностью изменять поведение сисетмы. Это настройки для ядра.

F1 Справка: Зависит от содержания. Она покажет различные разделы в зависимости от активного элемента экрана загрузки. Также доступно описание системы помощи.

F2 Видео режим: Выбор различного разрешения экрана используемого во время установки. Если у вас есть проблемы с графической установкой, вы можете использовать текстовый режим.

F3 Источник Установки: Выбор источника установки.

F4 Язык: Установка языка и раскладки клавиатуры используемой загрузчиком.

F5 Обновление драйвера: Для новых компьютеров, для установки системы может понадобиться обновленные драйвера.

Использование Системы помощи

Он-лайновая помощь загрузчика зависит от регистра букв. Она предоставляет информацию в зависимости от выбранного пункта в меню, или если вы редактируете параметр загрузки, она пытается найти информацию о параметре, под которым находится курсор.

Кнопки навигации

Up Arrow: подсвечивет предыдущую ссылку
Down Arrow: подсвечивает следующую ссылку
Left Arrow, Backspace: возврат к предыдущему разделу
Right Arrow, Enter, Пробел: переход по ссылке
Page Up: пролистывает на одну страницу назад
Page Down: пролистывает на одну страницу вперед
Home: переход к начальной странице
End: переход к конечной странице
Esc: выход из помощи

Обновление драйвера

Если вам необходим драйвер с обновлениями расположенный на гибком или лазерном диске, нажмите F5. Загрузчик попросит вас вставить соответствующий диск после загрузки ядра.

Драйвер с обновлениями это обычно дискетт с новой версией драйверов или исправлениями которые необходимы во время установки.

Загрузка с жесткого диска

Выберите Загрузить установленную ОС OS для загрузки системы установленной на ваш локальный жесткий диск. Индентификатор устройства первого жесткого диска предоставляется BIOS компьютером.

Используйте это если забыли вытащить CD или DVD из дисковода и хотите загрузить компьютер с жесткого диска.

Установка -- ACPI отключено

Иногда очень современное оборудование требует, чтобы ACPI управляло прерываниями. ACPI полностью заменяет старую APM систему.

Выберите Установка -- ACPI отключено, если у вас есть проблемы во время загрузки ядра. Известны проблемы с ACPI когда машина:

* ядро зависает при загрузке
* PCI карты не определяются или неправильно инициализируются

Параметры загрузки

Доступны два типа параметров загрузки. Во первых, параметры которые влияют на установку. Во вторых, есть параметры ядра. Некоторые наиболее используемых параметров:

a) параметры установщика

install -- выбор источника установки
network options -- сетевые параметры
vnc options -- параметры для установки через VNC

b) параметры ядра

splash -- включение/выключение заставки
apm -- переключение режима управления питанием
acpi -- расширенная конфигурация и интерфейс питания
ide -- управление IDE системой

Параметры установщика: install

По умолчанию, ищется CD-ROM как источник установки. Для сетевой установки, выберите параметр install. Возможные протоколы установки
* FTP
* NFS
* HTTP
Синтакс использования такой же как обычный URL. Для примера, если ваш сервер находится 192.168.0.1 и вы хотите осуществить NFS установку из каталога /install на этом сервере, укажите источник в следующем виде:

install=nfs://192.168.0.1/install

Сетевая карта будет также настроена при помощи dhcp или вы должны указать параметры сами как описано в сетевые параметры.

Параметры Ядра: splash

Экранная заставка это картинка показываемая при загрузки системы.

splash=0
Экранная заставка отключена. Это полезно при использовании очень старых мониторов или если есть какая-либо ошибка.

splash=verbose
Экранная заставка включена, однако сообщения загрузки и ядра по прежнему показываются.

splash=silent
Экранная заставка включена, никакие сообщения не показываются. Вместо сообщений показан индикатор выполнения загрузки.

Параметры установщика: network options

Возможно настроить стевой интерфейс прямо сейчас. Сетевое оборудование будет обнаружено позже при помощи YaST2. Минимальный набор параметров для настройки вашей сетевой карты состоит из IP адреса хоста и сетевой маски. Для примера:

hostip=192.168.0.10 netmask=255.255.255.0

или в более короткой форме:

hostip=192.168.0.10/24

Если вы указали сетевую установку и не указали оба этих параметров, установщик попытается настроить сетевой интерфейс при помощи dhcp. Если вам нужен шлюз по умолчанию, укажите его при помощи параметра gateway. Для примера:

gateway=192.168.0.8

Параметры Ядра: apm

APM одна из двух стратегий управления питания, используемых в современных компьютерах. Она в основном используется с лаптопами для таких функций как приостановление диска, но она таже может быть использована для выключения компьютера после выключения питания. APM зависит от правильной работы BIOS. Если BIOS поврежден, APM может быть использован с ограничением или вообще недопустить включение компьютера. Однако он может быть отключен при использовании параметра

apm=off -- выключает полностью APM

Некоторые очень новые компьютеры могут предоставляеть дополнительные параметры ACPI.

Параметры Ядра: acpi

ACPI (Расширенная конфигурация и интерфейс питания) это стандарт который определяет интерфейс управлением питанием и конфигурацией между операционной системой и BIOS. По умолчанию, acpi включен если обнаруженный BIOS моложе чем 2000. Есть несколько общих параметров для управления действиями ACPI:

pci=noacpi -- не использовать ACPI для управления PCI прерываниями
acpi=oldboot -- активируется только часть ACPI которая применима к загрузке
acpi=off -- выключает полностью ACPI
acpi=force -- включает ACPI даже если ваш BIOS датирован до 2000 года

На новых компьютерах, он заменяет старую apm систему.

Параметры Ядра: ide

IDE в отличии от SCSI, широко используется в настольных системах. При наличии проблем связанных IDE системами, используйте параметр ядра:

ide=nodma -- выключает режим dma для IDE дисков
