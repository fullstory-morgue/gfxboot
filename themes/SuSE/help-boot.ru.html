Использование Системы помощи

Он-лайновая помощь загрузчика зависит от регистра букв. Она предоставляет информацию в зависимости от выбранного пункта в меню, или если вы редактируете параметр загрузки, она пытается найти информацию о параметре, под которым находится курсор.

Кнопки навигации

Up Arrow: подсвечивет предыдущую ссылку
Down Arrow: подсвечивает следующую ссылку
Left Arrow, Backspace: возврат к предыдущему разделу
Right Arrow, Enter, Пробел: переход по ссылке
Page Up: пролистывает на одну страницу назад
Page Down: пролистывает на одну страницу вперед
Home: переход к начальной странице
End: переход к конечной странице
Esc: выход из помощи

Параметры загрузки

splash -- включение/выключение заставки
apm -- переключение режима управления питанием
acpi -- расширенная конфигурация и интерфейс питания
ide -- управление IDE системой

Параметры Ядра: splash

Экранная заставка это картинка показываемая при загрузки системы.

splash=0
Экранная заставка отключена. Это полезно при использовании очень старых мониторов или если есть какая-либо ошибка.

splash=verbose
Экранная заставка включена, однако сообщения загрузки и ядра по прежнему показываются.

splash=silent
Экранная заставка включена, никакие сообщения не показываются. Вместо сообщений показан индикатор выполнения загрузки.

Параметры Ядра: apm

APM одна из двух стратегий управления питания, используемых в современных компьютерах. Она в основном используется с лаптопами для таких функций как приостановление диска, но она таже может быть использована для выключения компьютера после выключения питания. APM зависит от правильной работы BIOS. Если BIOS поврежден, APM может быть использован с ограничением или вообще недопустить включение компьютера. Однако он может быть отключен при использовании параметра

apm=off -- выключает полностью APM

Некоторые очень новые компьютеры могут предоставляеть дополнительные параметры ACPI.

Параметры Ядра: acpi

ACPI (Расширенная конфигурация и интерфейс питания) это стандарт который определяет интерфейс управлением питанием и конфигурацией между операционной системой и BIOS. По умолчанию, acpi включен если обнаруженный BIOS моложе чем 2000. Есть несколько общих параметров для управления действиями ACPI:

pci=noacpi -- не использовать ACPI для управления PCI прерываниями
acpi=oldboot -- активируется только часть ACPI которая применима к загрузке
acpi=off -- выключает полностью ACPI
acpi=force -- включает ACPI даже если ваш BIOS датирован до 2000 года

На новых компьютерах, он заменяет старую apm систему.

Параметры Ядра: ide

IDE в отличии от SCSI, широко используется в настольных системах. При наличии проблем связанных IDE системами, используйте параметр ядра:

ide=nodma -- выключает режим dma для IDE дисков
